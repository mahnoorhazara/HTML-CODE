Q#1: What is HTML stands for ?

Ans: HTML stands for "Hyper Text Markup Language".

Q#2: What is HTML and its uses ?

Ans: HTML (HyperText Markup Language) is the standard language used to structure and design web pages.
It defines the layout of content using tags like headings, paragraphs, images, and links, providing the foundation for displaying text, multimedia, and interactive elements.
HTML works alongside CSS for styling and JavaScript for dynamic functionality.
It enables the creation of forms, embedding media, and enhancing search engine optimization (SEO) with metadata.
In short, HTML is essential for organizing and presenting content on the web, forming the backbone of all websites and web applications.

Q#3: What are HTML elements ?

Ans: HTML elements are the building blocks of a webpage. Here are the key points in short:

1: Tags: HTML elements are defined by tags, e.g., <p>, <h1>, <a>.

2: Structure: Each element has a start tag, content, and an end tag (e.g., <p>Hello</p>).

3: Attributes: Elements can have attributes that modify their behavior or appearance (e.g., href, src, <a href="https://example.com">link</a>).

4: Content: The information or data inside the tags (e.g., text, images).

5: Self-closing Elements: Some elements don't have an end tag, e.g., <img>, <br>.

6: Categories: Elements represent text, links, images, tables, forms, and more.

7: Purpose: HTML structures and formats the content of a webpage.
